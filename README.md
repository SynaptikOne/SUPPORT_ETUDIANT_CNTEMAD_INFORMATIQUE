# Laboratoire d’Innovation Libre et Open Source – SynaptikOne

Bienvenue sur le dépôt **SUPPORT_ETUDIANT_CNTEMAD_INFORMATIQUE**, initiative de **SynaptikOne** pour fournir des corrigés et des ressources pédagogiques en informatique.

---

## À propos

SynaptikOne est un laboratoire **libre et open source**, totalement indépendant, qui accompagne tous les étudiants, qu’ils soient inscrits dans une université ou autodidactes.  
Notre mission est de fournir un environnement **ouvert et collaboratif** pour apprendre, expérimenter et innover librement.

---

## Structure du dépôt

Le dépôt est organisé par **session**, puis par **niveau / parcours**, puis par **matière**. Chaque matière contient les fichiers **sujet** et **corrigé**.  

Exemple :

Arborescnece:




Chaque session contient :  
- **L1, L2, L3, M1, M2** selon les niveaux et parcours.  
- Chaque parcours contient des matières (Algo, Math, Réseau, Sécurité, ML, etc.).  
- Chaque matière contient :  
  - `Sujet_*.pdf` : le sujet de l’examen ou de l’exercice.  
  - `Corrige_*.pdf` : le corrigé correspondant.
